Web Tutorials

Diese Reihe von Tutorials soll einen Überblick darüber geben, was passiert, wenn der Browser verwendet wird, um eine Website zu besuchen. Sie sollen dabei helfen, Web-Technologien besser zu verstehen und schätzen zu lernen.

  • Eine URL mit dem Browser aufrufen
  • HTML, CSS, JavaScript - Die Frontend-Bausteine

Ein besseres Verständnis der grundlegenden Prinzipien, wenn auch auf sehr abstrakter Ebene, erleichtert die Arbeit in Web-Projekten und das Gespräch mit Web-Entwicklern. Die Tutorials vermeiden es, sich in Details zu verlieren. Während sie ermutigen, sich der Komplexität der Dinge bewusst zu werden, ermutigen sie auch, die Details zu ignorieren.

Da Sie einen Browser verwenden, um diese Seite zu lesen, wird davon ausgegangen, dass Sie wissen, was ein Browser und was eine URL ist. Es wird ebenfalls davon ausgegangen, dass Sie wissen, dass ein Server ein Computer ist auf dem eine Software läuft, um über ein Netzwerk (z.B. das Internet) mit anderen Computern zu kommunizieren.
