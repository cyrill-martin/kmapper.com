Domain-Ontologien für die Geisteswissenschaften

e-editiones.ch

e-editiones.ch wurde von Cyrill Martin während des Projekts Nationale Infrastruktur für Editionen an der Universität Basel entwickelt und wird von kmapper weiterentwickelt.

Kern der Website ist die Bibliothek der geisteswissenschaftlichen Ontologien, die als Turtle-Dateien in einem eigenen GitHub Repository gepflegt werden. Für die Website werden die Ontologien automatisiert in verschiedene RDF-Formate und HTML umgewandelt. e-editiones.ch unterstützt Content Negotiation, um spezifische Formate maschinell anzufragen und erfüllt die Kriterien von 5 Star Linked Open Data.

StackJekyll · Python · XSL · Docker

Themen im interdisziplinären Kontext visualisiert

kmapper.org

Die Bewältigung komplexer Probleme erfordert interdisziplinäre Forschung und eine offene, einfach zu navigierende Wissenslandschaft. kmapper ist so ein Knowledge Mapper.

Mit kmapper.org suchen Sie im DOAJ indexierte Artikel und die Ergebnisse werden entsprechend ihrer Relevanz und wissenschaftlicher Disziplin visualisiert.
